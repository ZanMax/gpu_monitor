# GPU Monitor
GPU VRAM, Core, Junction temperatures reader (GDDR6/GDDR6X) for NVIDIA GPUs

## Prerequisites

```bash
sudo apt update
sudo apt install build-essential gcc libpci-dev nvidia-utils-dev
```

## Build and Install

```bash
make
```

```bash
sudo make install
```
